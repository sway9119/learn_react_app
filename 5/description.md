## styledComponents

コンポーネント内に CSS を記述し、className にクラスを指定するのではなく、スタイルを適用したコンポーネントを定義する方法。

## Ecomtion

```
// インストール
$ npm install @emotion/react @emotion/styled
```

スタイル変数の定義時とタグの中にも使用し指定することで CSS を適用していく方法。
以下の全てに似たような書き方が用意されている。

- Inline Styles
- Styled JSX
- styled components

## Tailwind CSS

// インストール
$ npm install -D tailwindcss
